CREA TU PROPIA PAGINA WEB

Fecha de inicio: 15 de marzo de 2026

Categoría: Desarrollo

Estado: En Curso

Objetivos del Proyecto

Este proyecto busca desarrollar una plataforma educativa que permita a los estudiantes aprender los fundamentos de HTML y CSS de manera interactiva. El objetivo es que cualquier usuario pueda crear su primera página web siguiendo una guía clara y estructurada.

Además, se busca implementar buenas prácticas de programación, como el uso correcto de etiquetas semánticas y la organización de archivos en carpetas, facilitando el mantenimiento y escalabilidad del sitio a futuro.
